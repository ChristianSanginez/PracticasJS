How to use the pokedex?

Select one kind of search, the search types are by name, by type or by id

  • 1) To use the search by name, you need to enter the name of a Pokémon. If that name exists, the Pokémon will appear.
  • 2) To use the search by type, you need to enter a number. If that number exists, a Pokémon of that type will appear randomly.
  • 3) To use the search by id, you need to enter a number. if that number exists, the Pokémon with that id will appear.
